Démarrer

Pour bien démarrer avec le serveur PolyPeer, il faut s'assurer d'avoir la configuration minimale requise :

  • Un serveur dédié sous Linux (noyau >= 2.6)
  • Des clients sous Linux ou Windows (XP)

Pour lancer le serveur, rien de plus simple, prenez un invite de commande Bash et entrez ./polypeer.

Après une phase d'initialisation, le serveur sera prêt à déployer les fichiers !

Pour lancer le client, vous avez juste aussi à lancer le client ./ppclient en spécifiant ou non l'option de démon/service.

Créer un déploiement

Afin de créer un nouveau déploiement vous devez vous rendre sur Nouveau déploiement.

Il va falloir remplir les champs demandés avec des informations valides :

  • Le nom du déploiement doit être sans majuscules
  • Le chemin d'accès au fichier sur le serveur doit être valide (et exister)
  • Le chemin de destination chez le client doit lui aussi exister

Une fois cela fait, il vous reste à choisir au moins un ordinateur dans la topologie du réseau pour déployer le fichier !

Actions sur les déploiements

Tout au long de vos déploiements, vous pourriez être amené a en supprimer ou stopper certains.

Ainsi, vous disposez d'un bouton d'arrêt général des déploiements sur la page de visualisation des déploiements. Cela aura pour effet de mettre en pause les déploiements en cours chez les clients. Vous pourrez reprendre à n'importe quel moment les déploiements en recliquant dessus.

Lorsque vous cliquez sur un déploiement, deux autres actions sont aussi disponibles. Vous pouvez choisir de mettre en pause le déploiement ou de le supprimer.

Attention ! Supprimer un déploiement est irréversible.
