Description détaillée

Fonction amie pour accéder facilement aux attrs du webserver

Fonction d'handle des events sur le serveur web pour faire simple, si cette fonction ne retourne pas null cela veut dire que la requête a été traité par nous même, si elle retourne null on laisse mongoose la gérer (il va alors tenter d'y répondre lui même

Paramètres:
mg_event evenement déclenché par une requête
mg_connection* connexion courante
mg_request* contient tous les paramètres de la requête reçue
Renvoie:
processed requête prise en charge ou non
