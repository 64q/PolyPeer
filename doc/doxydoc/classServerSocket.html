Description détaillée

Classe qui permet l'écoute de connexions tcp entrantes. On choisit un port d'écoute grace au constructeur puis on récupère les connexions entrantes grace à la fonction accept() qui va renvoyer une socket permettant de communiquer avec l'ordinateur connecté.

Socket * ServerSocket::accept (  ) 

Méthode bloquante permettant la réception des connexions entrantes. La méthode bloquera l'exécution du reste du code jusqu'à ce qu'elle reçoive une connexion. Elle renverra alors une socket permettant de recevoir des données et renvoyer des données à l'entité qui a demandé la connexion.

Renvoie:
Socket objet permetant de communiquer avec l'entité qui a demandé la connexion.(voir Socket)
